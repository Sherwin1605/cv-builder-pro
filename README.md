# CV Builder Pro 🚀

Ứng dụng tạo CV chuyên nghiệp, hiện đại và dễ sử dụng được xây dựng bằng React, Next.js và Tailwind CSS.

## ✨ Tính năng nổi bật

- **Nhiều mẫu CV chuyên nghiệp:** Hiện đại, Tối giản, Sáng tạo và Chuyên nghiệp.
- **Chỉnh sửa thời gian thực:** Xem thay đổi ngay lập tức khi bạn nhập dữ liệu.
- **Quản lý ảnh đại diện:** Hỗ trợ tải ảnh từ máy tính hoặc sử dụng URL, có tính năng xóa ảnh linh hoạt.
- **Tùy chỉnh màu sắc:** Thay đổi màu chủ đạo của CV chỉ với một cú click.
- **Xuất PDF:** Tích hợp tính năng in và xuất file PDF chất lượng cao.
- **Giao diện thân thiện:** Thiết kế hiện đại, mượt mà với hiệu ứng chuyển động từ Framer Motion.

## 🛠 Công nghệ sử dụng

- **Frontend:** React 19, Vite, Tailwind CSS 4.
- **Icons:** Lucide React.
- **Animations:** Framer Motion.
- **PDF Export:** React-to-print.
- **Backend (Tùy chọn):** Supabase (Auth & Database).

## 🚀 Hướng dẫn cài đặt

1. **Clone repository:**
   ```bash
   git clone https://github.com/HuyHoagnn/cv-builder-pro.git
   cd cv-builder-pro
   ```

2. **Cài đặt dependencies:**
   ```bash
   npm install
   ```

3. **Chạy ứng dụng ở chế độ phát triển:**
   ```bash
   npm run dev
   ```

4. **Xây dựng bản sản xuất:**
   ```bash
   npm run build
   ```
